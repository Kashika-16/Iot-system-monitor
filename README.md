# IoT Sensor Monitoring System

A full IoT pipeline using ESP32 + MQTT + Python backend + Grafana dashboard.

## Hardware
- ESP32
- DHT22 sensor
- USB cable
- WiFi network

## Software Stack
- MQTT (Mosquitto)
- Python subscriber
- Grafana (dashboard)
- InfluxDB or SQLite (storage)
